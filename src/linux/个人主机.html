Rockylinux8分区 ​

  • / (根目录)：系统的根挂载点，所有的文件和目录都从这里开始。对于一般用途的服务器，给根目录分配 20GB 到 50GB 的空间通常是足够的。如果计划安装大量的软件或服务，可能需要更多空间。
  • /boot：包含启动加载器和一些启动时需要的文件。分配给它的空间不需要很大，一般 1GB 左右即可。
  • swap：交换分区用作当物理内存用完时的备用内存。传统的做法是将交换空间设置为物理内存（RAM）的两倍，但现在随着物理内存量的增加，如果你的系统有 8GB 或更多的 RAM，通常分配与物理内存相等或一半的 swap 空间就足够了。
  • /home：用户的个人文件存放在这个目录下。将 /home 放在单独的分区可以在重装系统时保留用户数据。其大小取决于你计划有多少用户和他们需要的空间。
  • /var：对于运行大量日志、邮件服务器或数据库服务的服务器，/var 应该考虑单独分区，因为这些服务会在 /var 目录下生成大量数据。大小依服务而定。

[EPEL]

docker 安装 ​

bash
cat > /etc/yum.repos.d/docker-ce.repo <<EOF
[docker-ce]
name=Docker CE USTC
#baseurl=http://mirrors.ustc.edu.cn/docker-ce/linux/rhel/8/x86_64/stable/
baseurl=http://mirrors.ustc.edu.cn/docker-ce/linux/centos/8/x86_64/stable/
gpgcheck=0
EOF

dnf install -y docker-ce-24.0.0 docker-ce-cli-24.0.0

cat > /etc/docker/daemon.json << EOF
{
  "registry-mirrors": ["https://qmaqgl5w.mirror.aliyuncs.com"]
}
EOF

systemctl enable --now docker
